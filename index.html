⚙️ Quick Setup Guide

1 Install Prerequisites

Ensure you have Node.js (v16+), npm, and MySQL (v8+) installed.

2 Database Setup
mysql -u root -p < backend/config/database.sql

This creates the database, tables, and sample data.

3 Backend Setup
cd backend npm install cp .env.example .env # Edit .env with your database credentials npm run dev

Backend will run on http://localhost:5000

4 Frontend Setup
cd frontend npm install npm run dev

Frontend will run on http://localhost:3000
